# `components/ui/` — Base UI Components

## Purpose
Primitive UI components used as building blocks throughout the application. These are the lowest-level, most reusable components (similar to shadcn/ui components).

## Current Files
| File | Description |
|---|---|
| `theme-toggle.tsx` | Dark/light mode toggle button |
| `language-switcher.tsx` | Locale selector dropdown |

## Future Files (shadcn/ui style)
- `button.tsx` — Button with variants (primary, secondary, destructive, ghost, link)
- `input.tsx` — Text input with label and error state
- `textarea.tsx` — Multi-line text input
- `select.tsx` — Dropdown select
- `checkbox.tsx` — Checkbox input
- `switch.tsx` — Toggle switch
- `dialog.tsx` — Modal dialog
